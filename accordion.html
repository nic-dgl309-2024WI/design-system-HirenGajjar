Accordion

Notation

Accordion in web development is a user interface component that allows content to be collapsed or expanded with a click, revealing or hiding additional information.

It typically consists of a series of collapsible panels, where only one panel is expanded at a time, conserving space and providing a streamlined user experience. Accordions are commonly used for FAQs, menus, and other content-heavy sections on websites.

Example

Question?

+

Answer

HTML Structure

<div class="accordion-wrapper">
  <div class="accordion-question">
    <p class="accordion-question__text">Question ?</p>
    <p class="accordion-question__icon">+</p>
  </div>
  <div class="accordion-answer">
    <p class="accordion-answer__text">Answer.</p>
  </div>
</div>

JavaScript

let acc = document.querySelectorAll('.accordion-class');
   acc.forEach ( accordian => {
     let accIcon = accordian.querySelector('.accordion-question');
     let accAns = accordian.querySelector('.accordion-answer');
     accordian.addEventListener('click', () => {
     acc.forEach(otherAccordion => {
          if (otherAccordion !== accordian) {
               otherAccordion.querySelector('.faq-acc__ans').classList.remove('expand');
               otherAccordion.querySelector('.faq-acc__que-icon').textContent = '+';
          }
     });
     accAns.classList.toggle('expand');
     accIcon.textContent = accIcon.textContent === '+' ? '-' : '+';
   });
});
