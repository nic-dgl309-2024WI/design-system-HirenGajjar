Spacing

Notation

The spacing layout classes are named using the format .l-{property}{sides}-{size}.

The sides are named using shorthand as shown in the example. ml is margin-left or pt is padding-top .

Similarly, the size applied to each side starts at a quarter of a unit, increasing with each step. In other words, '050' represents 0.5rem (8px), and '125' denotes 1.25rem (20px).

Horizontal spacing

Horizontal spacing layout classes are useful for applying equal spacing along the x-axis of an element.

The sides are named using shorthand alphabet. 'x' (on x-axis). mx is margin-left & margin-right or px is padding-left & padding-right .

The key point here is that the same value is applied to both sides..
