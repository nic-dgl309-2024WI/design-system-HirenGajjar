Breadcrumbs

Notation

A breadcrumb is a textual map that helps users know their location on websites and aids in navigating back and forth within the site,
generally place above page titles

A breadcrumb trail has four main parts:
Home ,Page link, Separator,and Current page.

Example

Home
Home > Current Page
Home > Page Link 1 > Current Page
Home > Page Link 1 > Page Link 2 > Current Page
Home > ... > Current Page

Usage

For Hierarchical Website

HTML Structure (add to each page.)

<div id="breadcrumb"></div>

JavaScript

function generateBreadcrumb() {
var url = window.location.pathname;
var urlSegments = url.split('/').filter(segment => segment !== '');
var breadcrumbHTML = '<a href="/">Home</a>';
var currentUrl = '/';
for (var i = 0; i < urlSegments.length; i++) {
currentUrl += urlSegments[i] + '/';
var segment = urlSegments[i].charAt(0).toUpperCase() + urlSegments[i].slice(1);
breadcrumbHTML += ' / <a href="' + currentUrl + '">' + segment + '</a>';
}
document.getElementById('breadcrumb').innerHTML = breadcrumbHTML;
}
window.onload = generateBreadcrumb;

For Single Page Website

HTML Structure

<div id="breadcrumb">
<a href="index.html">Home</a>
<span class="separator"> > </span>
<a href="#" id="#sectionNumber">Section</a>
<span class="separator"> > </span>
<a href="#" id="#sectionNumber">Current Section</a>
< /div>

function copyCode() {
var code = document.querySelector('.breadcrumbs').innerHTML;
navigator.clipboard.writeText(code)
.then(() => alert('Code copied successfully!'))
.catch(err => console.error('Failed to copy code: ', err));
}
